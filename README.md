# you-film

## Project setup
```
1. git clone https://github.com/WebFIow/you-film.git
2. cd you-film
3. npm install
4. create .env file with secret info
5. npm run serve
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
